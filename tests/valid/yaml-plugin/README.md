# YAML Test Plugin

This is a test plugin using **YAML format** for the manifest file.

## Features

- ✅ YAML format manifest
- ✅ All required fields present
- ✅ Semantic versioning
- ✅ Proper command definition

## Installation

```bash
<command_launcher_cli> package install --file <url>
```

If your CLI is named `cdt`:

```bash
cdt package install --file <url>
```

## Usage

```bash
yaml-test --verbose
```
