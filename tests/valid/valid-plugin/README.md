# Valid Test Plugin

This is a valid test plugin for testing the Command Launcher Package Action.

## Features

- ✅ Valid manifest format
- ✅ All required fields present
- ✅ Semantic versioning
- ✅ Proper command name format

## Installation

```bash
cola install valid-test
```

## Usage

```bash
valid-test --help
```

## Testing

This plugin is used for testing the validation, packaging, and documentation generation features of the Command Launcher Package Action.
